Auto Bake is a capable but very easy to use texture baker add-on with the goal to provide a fully automated, seamless workflow while ensuring simplicity and full control. It can also export textures, create a baked version of the objectss with automatic texture setup, export the baked objects, supports advanced materials, and a lot more!
WORKFLOW

Set up a bake list for the textures you want to bake, then simply select the objects in the viewport you want to bake, and the bake is ready to be started! After the bakes are finished, a baked version of the objects are created with the baked textures automatically applied to them. The baked textures and objects can then be exported with a single button with your custom export settings.

EASE OF USE

Auto Bake is designed for both beginners and advanced users in mind. The panels and their content are thoughtly organized, the buttons and options are enchanced with tooltips to help knowing how to use them. The documentation explains the buttons and options in great details and with examples.

TEXTURE MAPS

A wide variety of textures can be baked with Auto Bake, including shader sockets, multiresolution modifier data, Blender's default bake passes, and a set of miscellaneous bake types for channel packing textures, baking pointiness map of the object, or baking bevel normal, and more!

MATERIAL SUPPORT

Auto Bake is designed both for the very basic single material per objects and for the advanced setups with multiple materials per object. The materials can also vary from a single shader node to the complex node setups with mixing multiple shader nodes and using node groups.

Node Setup Example

Although the texture maps are designed around the Principled BSDF node, it also supports all the other shaders, some are to a fully and some to partially extent, depending on the shader's input sockets. If a socket has a matching pair with one of the Principled BSDF's socket, then it can be baked using the same texture map as for the Principled BSDF node.

Fully & Partially Supported Examples
BAKED VERSIONS

By enabling a single option, Auto Bake will create a final material for the baked textures, and automatically set them up, connecting them to their corresponding input socket of the shader node, with the nodes automatically position and aligned.

Automatic Texture Setup

With another option enabled, Auto Bake will not only set up the materials and textures, but it will also create a baked version of all the baked objects, applying their own final materials. Additionally, then the original object can be completely replaced with the baked object using the confirm button.

BAKE PROGRESS

Keep using Blender! When baking textures are being baked in the background, so Blender won't freeze. The progress of the bakes can be seen in the Queue panel.

INSPECT TEXTURES

Just as the textures can be previewed before baking them, the textures can also be inspected on all the objects after baking them. With a single option the texture inspection is enabled and the texture itself is projected onto its object

Texture Inspection
EXPORT

Auto Bake gives the option to export either or both the baked textures and the objects with a single button, with your own custom export settings. You can choose different file format both for the textures and objects.

Options are also provided for folder control. Use of master folders can be enabled to place both the exported object and its textures in. Subfolders can also be enabled to organize the textures by type.

WORKSPACES

You have the option to choose the workspace you want to use Auto Bake from. You can use it either from the Render properties where you would normally bake, or from the 3D viewport's sidebar, or even from the Shader Editor's sidebar. The decision is yours, select the one you prefer!

SOURCE TO TARGET

Baking textures from source objects is not only supported but also has a few useful functions, including cage object generator, extrusion visualizer and more!

You can choose between the default method of selecting the target and source objects in the viewport, or you can use Auto Bake's advanced method, with you can set up multiple target objects with their own source objects and settings each.

This method will try to find the cage and all the source objects, based on the target object's name, settings them up automatically.
