Bake a wide range of texture maps, preview them before baking, inspect the baked textures, export the baked textures and objects, apply the baked textures to the objects, create a baked version of the objects and materials. Bake UDIM and atlas textures, apply anti-aliasing, channel pack the textures. Generate new UVs, choose a target UV, remove unused UV maps, bake from source to target objects, generate cage objects, visualize extrusion, create and use your custom presets, and choose your workspace.

WORKFLOW

Set up a bake list for the textures, select the objects in the viewport you want to bake, then start the bake, and follow the progress in the bake queue! Create a baked version of the objects with applying the baked textures to them. Export the baked textures and objects with your own export format and settings.

EASE OF USE

Designed for both beginner and advanced Blender users, with barely anything needed to change to bake and use it, but with high custimibility options. Buttons and options are structured and categorized into panels, and suited with tooltips on what they do and how they work, and with texts alerts when something is wrongly set. A provided documentation that guides you through each steps, with great details and examples.

PREVIEW TEXTURES

Preview the textures maps before baking them, by selecting the objects to preview, and enabling it from the bake list. The preview is avilable for all the shader and miscellaneous bake maps.

Texture Preview

Customize the texture maps with adjustable properties and view the changes in real-time while previewing it. Different texture maps have different amount and different properties options to change.

Real-time Preview Update
MATERIAL SUPPORT

Bake both the very simple materials, the same way as the complex material setups with multiple shaders mixed and added together. Node groups can also be used both simple or nested node groups. Objects can also have multiple materials assigned to them.

Node Setup Example

Use shader nodes other than the Principled BSDF. While the shader texture maps are designed around the Principled BSDF, other shader nodes are also supported to bake, some are partially while some are fully, depending on the shader's input sockets.

Fully & Partially Supported Examples

When a shader node has matching input sockets with the Principled BSDF node, the matching sockets then can be baked iwth the the same shader texture map as the Principled BSDF node.

BAKED VERSION

Convert the baked objects and their textures into a baked and final version, with a copy of the original object with a new final material assigned to it, and the baked textures automatically applied to it, with the texture nodes aligned to each other and connected to their corresponding shader sockets.

Automatic Texture Setup

This process will result a complete conversion from the original unbaked object and its materials, to a baked version of it.

BAKE PROCESS

Keep using Blender while the textures are being baked, Blender won't freeze! Follow the bake progress of the textures in the queue panel and cancel or pause the bake process at any time.

Bake Queue
INSPECT TEXTURES

Overview the results after the textures are baked. By enabling the texture inspection option directly from the queue panel, the selected textures in the queue are projected onto their corresponding objects.

Texture Inspection
EXPORT

Texture & Object Export

Control the folders what to create for the exported objects and textures and what not. Set wether to create a master folder, or subfolders for different texture map types.

UV TOOLS

Set any of the UV maps of the object to be the target UV and to bake the textures to. It opens the possbility to use different UV maps in the shaders, or to rebake textures from one UV to another.

UV Maps
Smart UV Project

The unused UV maps can also set to be removed from the final objects, only keeping the one the textures were baked to. Objects without a UV map are will have one generated via Blender's Smart UV Project.
